AIDE

COMMANDES

git init

creer un depot dans le dossier cible

git status

donner le status du depot

git add nom_de_fichier

ajouter tel fichier au depot

git add .

Ajouter tous les fichiers modifié

git commit

créer un commit a partir de la zone d'index

git commit -m"#n message"

commit avec un message ou "n" est le numero de l'issue

git diff –cached fichier

affiche les modifications indexé

git log

historique des commit

git log -n 2

affiche les 2 derniers commits(on peut remplacer le chiffre 2 par le nombre de commit qu'on veut voir)

git show sha-1

affiche le contenu du commit correspondant

git show tag

meme que precedent

git checkout sha-1

se deplacer sur le commit sha-1

git checkout master

revenir au dernier commit (tag master)

git tag --delete nom

supprimer le tag

git tag nom -m"description"

ajouter un tag avec description

git tag

lister les tags

git clone [adresse] [folder name]

clone un depot git

git remote add "nom" https://.............

créer un remote

git remote -v

afficher les remote

git push -u [remote] [branche]

pousser un remote ds une branche

git push [remote] [tag]

pousser un tag sur le remote

git push [remote] -tags

pousser tt les tags sur le remote(si pas spécifié prend par default origin)

git fetch

recuperer info sur server distant

git pull

ecrire les info ds depot local

git blame [fichier]

affiche le contenu d'un fichier + des infos

git blame -L 10,20 [fichier]

affiche le contenu du fidchier de la ligne 10 a 20

git blame -L 10,+4 [fichier]

affiche le contenu du fichier a partir de 10 seulement 4 lignes

git stash save "description"

mettre dans la zone de stash les modifications du 'workspace' 

git stash list

listing des modifications dans 'stash'

git stash show'index'

description des modifications du stash

git stash pop 'index'

injecter les modifications

git pull --rebase

faire un rebase au lieu d'un merge(plus propre)

git branch "nom"

créé une nouvelle branche

git branch

liste les differentes branches en local

git branch -a

liste les differentes branch distante

git branch -d "nom de la branche"

supprime la branche

git push origin --delete "nom"

supprimer branche distante

git cherry-pick sha-1

copier commit d'une branche a une autre

retour accueil
